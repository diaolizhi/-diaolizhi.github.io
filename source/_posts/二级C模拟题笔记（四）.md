- 计算顺序类型
a+=a-=a\*a
先算 a\*a=9，然后 a-=9，a 就等于 -6。
**注意这里 a 等于 -6 了！！！**

- 若有以下程序段
char str[4][12]={ "aa","bbb","ccccc","d" } , \*strp[4];
int i;
for( i = 0; i< 4; i++ ) strp[i] = str[i];
不能正确引用字符串的选项是 
A) str[0] 
B) strp 
C) strp[3] 
D) \*strp 
**注意看它们是不是指向字符数组**

- 字符指针传值问题
虽然实参是个指针，但形参也是个指针，相当于创建一个新的指针变量，然后它的值是和实参一样，改变它的值不会改变原来字符串。
